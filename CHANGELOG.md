# Changelog

## [1.0.0] — June 2026

### Initial public release

- IBM WS-BPEL 2.0 → Orkes Conductor JSON conversion
- Support for all core BPEL constructs: sequence, flow, if/elseif/else, while, repeatUntil, invoke, receive, reply, assign, throw, catch, compensate, scope
- Visual flow diagram generation
- Traffic-light migration assessment (green / amber / red)
- PDF export of assessment report
- One-click deploy to Orkes Developer Cloud
- REST API with X-Api-Key authentication
- MCP server (HTTP transport) — compatible with Claude, Cursor, Windsurf
- GitHub login with 50 free conversion credits
- Credit purchase for additional conversions
- Four example BPEL files: loan approval, credit card provisioning, income verification, communication
